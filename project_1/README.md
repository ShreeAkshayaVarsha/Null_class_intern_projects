Dynamic Knowledge Base Chatbot (FAISS + Python)

This project implements a **dynamic knowledge-based chatbot** using **FAISS vector database**.  
The chatbot can automatically incorporate **new information** by re-ingesting data, without retraining any model.

This project is beginner-friendly and demonstrates the core idea behind **Retrieval-Augmented Generation (RAG)** systems.


Project Objective

To build a chatbot that:
- Stores knowledge in a **vector database**
- Retrieves answers using **semantic similarity**
- Can **expand its knowledge base dynamically** over time


Technologies Used

- Python 3.x  
- FAISS (Facebook AI Similarity Search)  
- Sentence Transformers (`all-MiniLM-L6-v2`)  
- NumPy
